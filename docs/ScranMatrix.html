ScranMatrix

Wrapper around a sparse matrix allocated on the Wasm heap.

Constructor

new ScranMatrix(raw)

Source:
  • ScranMatrix.js, line 6
Parameters:
Name Type Description
raw Object

Raw matrix object created on the Wasm heap.

This should not be called directly by developers, who should instead use functions that initialize an appropriate matrix, e.g., initializeSparseMatrixFromCompressedVectors().

Obtain the identities of the rows of the matrix, assuming isReorganized returns true.

Parameters:
Name Type Attributes Description
options Object <optional>

Optional parameters.

Properties
Name Type Attributes Description
buffer Int32WasmArray <optional>
<nullable>

Buffer to extract into. If supplied, this should have length equal to numberOfRows().
Returns:

If buffer is not supplied, an Int32Array is returned containing the row identities. These represent the row indices in the original dataset.

If buffer is supplied, it is filled with the row identities. A reference to buffer is returned.

isReorganized()

Source:
  • ScranMatrix.js, line 112
Returns:

Boolean indicating whether the matrix contains a non-trivial organization of row identities. If true, the row identities should be extracted from identities; otherwise, the row identities are assumed to be consecutive increasing integers from 0 up to the number of rows.

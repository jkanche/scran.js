Class: LayeredSparseMatrix

LayeredSparseMatrix(raw)

Wrapper around a layered sparse matrix allocated on the Wasm heap. This permutes the rows to achieve a more memory-efficient representation.

Constructor

new LayeredSparseMatrix(raw)

Parameters:
Name Type Description
raw Object

Raw matrix object created on the Wasm heap.

This should not be called directly by developers. Rather it is called by functions that initialize a sparse matrix, e.g., initializeSparseMatrixFromCompressed().

Returns:

If buffer is not supplied, a Int32Array is returned containing the permutation vector. Otherwise, buffer is filled with the permutation vector and nothing is returned.

If restore = true, the permutation vector represents the permutation should be applied to the current rows to obtain the original row order. That is, for a permutation array x, x[i] is the current row index for the original row i.

If restore = false, the permutation vector represents the permutation that was applied to the original rows to obtain the permuted (i.e., current) row order. That is, for a permutation array x, x[i] is the original row index for the current row i.

Class: PCAResults

PCAResults(raw)

Wrapper for the PCA results on the Wasm heap.

Constructor

new PCAResults(raw)

Parameters:
Name Type Description
raw Object

Raw results allocated on the Wasm heap.

This should not be called directly by developers. Instead, PCAResults objects should be created by calling runPCA().
